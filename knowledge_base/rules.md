# TechNexus Community Rules

## Core Principles

Respectful communication, professional conduct, ethical innovation, inclusive participation, and a safe learning environment — across physical events, virtual sessions, WhatsApp groups, and all official channels.

## Prohibited Behavior

- **Spam**: repeated promotions, unsolicited links, mass forwarding
- **Unrelated promotions**: marketing unrelated products, self-promotion or job ads without organizer approval
- **Fake or misleading information**: false event claims, unauthorized voucher promises, impersonating organizers
- **Disrespectful conduct**: personal attacks, harassment, discrimination, gender bias, offensive language
- **Event disruption**: interrupting sessions, bypassing verification, using the community for personal monetary or social-media gains

## WhatsApp Group Policy

Groups are moderated and event-focused. Allowed: event updates, official announcements, confirmations, session materials, relevant tech discussions. Not allowed: viral forwards, political/religious debates, personal ads, non-tech content.

## Event Entry Compliance

Confirmed RSVP, event pass or email confirmation, QR code (if issued), WhatsApp group verification, and valid ID if required. For Microsoft-hosted/partner venues: follow venue security, check in 30 minutes prior, respect seat allocation, photography restrictions, and guest rules. Non-compliance may result in denied entry.

## Venue Conduct

At partner venues (e.g., Microsoft), attendees must follow the venue's Code of Conduct, security screening, professional behavior guidelines, and respect for staff. TechNexus does not override partner venue policies.
